# Movie_android
* edwith 부스트코스에서 진행한 영화 정보 어플리케이션 예제입니다.  
* 영화순위, 좋아요, 댓글 등의 기능을 구현하였습니다.
* Volley, Gson, Glide 등의 라이브러리를 사용하였습니다.
* 영화 포스터와 영화 예고편을 확인 할 수 있습니다.
